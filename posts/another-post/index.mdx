```python:title=solution.py
def good_solution(self, nums: list[int]) -> list[int]:
    length = len(nums)
    out = [1] * length
    pre = 1
    post = 1

    for i in range(len(nums)):
        out[i] *= pre
        pre *= nums[i]

        out[length - i - 1] *= post
        post *= nums[-(i+1)]

    return out

```
